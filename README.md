## 安装方式

大部分skill支持以下安装方式：

1. **Claude Code Plugin Marketplace** - 最推荐
2. **手动安装** - 复制到 `~/.claude/skills/`
3. **MCP服务器** - 通过MCP协议使用
4. **通用安装器** (OpenSkills、Agent Skills CLI)
